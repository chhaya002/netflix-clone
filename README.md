# Netflix Clone

A responsive Netflix landing page clone built using HTML and CSS.

## Preview

This project recreates the Netflix homepage interface with a modern and responsive design.

## Features

* Responsive Layout
* Hero Section
* Feature Sections
* FAQ Section
* Footer Navigation
* Netflix-inspired UI
* Clean HTML Structure
* Organized CSS Styling

## Technologies Used

* HTML5
* CSS3

## Project Structure
